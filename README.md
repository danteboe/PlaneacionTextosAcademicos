# PlaneacionTextosAcademicos

Presentación interactiva en Next.js sobre planeación de textos académicos (texto argumentativo) para el curso Experiencia y Construcción Social.

## Ejecutar en local

1. Instala dependencias:

```bash
npm install
```

2. Inicia el servidor de desarrollo:

```bash
npm run dev
```

3. Abre en el navegador:

```text
http://localhost:3000
```

## Contenido

- App de slides en `app/page.tsx`
- Datos de cada diapositiva en `app/slides.ts`
- Estilos visuales en `app/globals.css`

## Navegación durante la presentación

- Flecha derecha / tecla `D`: siguiente diapositiva
- Flecha izquierda / tecla `A`: diapositiva anterior
- `Home`: ir al inicio
- `End`: ir al final
